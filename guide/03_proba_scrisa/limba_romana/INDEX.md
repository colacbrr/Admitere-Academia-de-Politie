# Limba romana

**Rolul sectiunii**
Sectiunea ofera pregatire completa pentru itemii de limba romana: gramatica si sintaxa, morfologie, vocabular, cu accent pe explicatii aplicate si justificarea raspunsului corect.

**Capitole**
- Gramatica si sintaxa
- Morfologie
- Vocabular

**Metoda de parcurgere**
- Start: gramatica si sintaxa (structura frazei).
- Consolidare: morfologie (forme, acorduri, valori).
- Finalizare: vocabular contextual si capcane lexicale.

**Ce urmarim la invatare**
- Corectitudine normativa.
- Viteza de analiza in item grila.
- Argumentare clara a alegerii raspunsului.
- Retentie prin exercitiu repetat.
- Intelegerea raportului dintre norma academica, uz curent si registru contextual.
- Capacitatea de a justifica lingvistic raspunsul, nu doar de a-l intui.

**Standard de calitate pe capitol**
- Capitolul este considerat \"gata\" la prag minim 90%: definitii corecte, mecanism explicat, exemple validate, capcane frecvente si exercitii aplicate.
- Pentru recapitulare, recomandarea este ciclul 24h -> 7 zile -> 30 zile.
