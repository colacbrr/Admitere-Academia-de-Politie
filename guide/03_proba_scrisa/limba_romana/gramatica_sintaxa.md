# Gramatica si sintaxa

**Ce trebuie sa invete candidatul**
- Diferenta corecta dintre enunt, propozitie si fraza.
- Tipurile de raporturi sintactice: coordonare, subordonare, apozitie.
- Functiile sintactice centrale si secundare.
- Tipologia propozitiilor subordonate si markerii uzuali.
- Reguli de punctuatie aplicate sintactic, nu intuitiv.

**Ce trebuie sa pregateasca candidatul**
- Tabel complet cu intrebari-ghid pentru functii sintactice.
- Set de 100 de itemi pe subordonate (atributiva/completiva/circumstantiale).
- Fise de transformare: fraza -> propozitii componente + raporturi.
- Teste cronometrate de 20-30 minute pentru viteza si acuratete.

**Rezumat**
Gramatica si sintaxa sunt zona cu cel mai mare impact in punctajul de limba romana: cine recunoaste rapid structura frazei, rezolva corect atat itemii de analiza, cat si itemii de norma.

**Explicatie detaliata**
- Enuntul este unitatea de comunicare cu sens complet in context.
- Propozitia este unitate sintactica centrata pe un predicat.
- Fraza este ansamblul format din minimum doua propozitii aflate in raport de coordonare/subordonare.
- Raportul de coordonare leaga unitati de acelasi rang sintactic (copulativ, adversativ, disjunctiv, concluziv).
- Raportul de subordonare leaga o unitate dependenta de un termen regent.
- Subiectul exprima autorul logic al actiunii/starii.
- Predicatul verbal exprima actiune/stare prin verb predicativ.
- Predicatul nominal se formeaza din verb copulativ + nume predicativ.
- Atributul determina un termen nominal (substantiv/pronume).
- Complementul direct completeaza un verb tranzitiv si raspunde frecvent la "ce?"/"pe cine?".
- Complementul indirect exprima relatii de destinatar, posesie, interes etc. (intrebari de tip "cui?").
- Complementul circumstantial precizeaza imprejurari: timp, loc, mod, cauza, scop, concesie, conditie, consecinta.
- Subordonata atributiva determina un termen nominal din regenta si se introduce adesea prin pronume/adverbe relative.
- Subordonata completiva directa completeaza un verb tranzitiv din regenta.
- Subordonata completiva indirecta completeaza un termen regent prin relatie indirecta.
- Subordonata circumstantiala de timp raspunde la "cand?"; de loc la "unde?"; de mod la "cum?".
- Subordonata cauzala raspunde la "din ce cauza?"; finala la "cu ce scop?".
- Subordonata conditionala exprima conditia realizarii actiunii din regenta.
- Subordonata concesiva exprima o opozitie admisa fara anularea rezultatului regentei.
- Subordonata consecutiva exprima efectul rezultat din regenta.
- Punctuatia corecta se stabileste prin raport sintactic, nu prin pauza de respiratie.
- Virgula separa de regula propozitii coordonate/adversative si multe subordonate incidentale.
- In completive directe esentiale, virgula este adesea absenta (in functie de constructie).
- In subordonatele explicative incidentale, izolarea prin virgule este frecventa.
- Pentru examen, metoda practica: identifici predicatele -> delimitezi propozitiile -> gasesti regenta -> stabilesti raportul -> aplici punctuatia.
- Capcana frecventa: confundarea atributivei cu completiva directa din cauza pronumelor relative.
- Capcana frecventa: clasificarea dupa "cum suna" in loc de clasificare dupa functie sintactica reala.
- Capcana frecventa: acord gresit in predicat nominal la plural colectiv.
- Exercitiu recomandat: delimiteaza 20 de fraze complexe si eticheteaza raporturile sintactice in maxim 15 minute.
- Exercitiu recomandat: reformuleaza 10 fraze prin schimbarea tipului de subordonata (exemplu: cauzala -> finala).
- Repetitia activa pentru retentie: 20 minute/zi pe delimitare de fraza + 20 minute/zi pe subordonate.
- Acordul gramatical se verifica pe lant: subiect -> predicat -> elemente atributive dependente.
- In frazele lungi, predicatul se identifica mai sigur prin cautarea verbelor personale, nu prin intuitie de pauza.
- Marcatorii de subordonare ("ca", "sa", "daca", "cand", "deoarece", "incat") trebuie validati prin functie, nu doar prin forma.
- Complementul prepozitional nu este echivalent automat cu complementul indirect; functia depinde de regent.
- In analiza sintactica, ordinea recomandata este: predicate -> delimitare propozitii -> raporturi -> functii -> punctuatie.
- Capcana frecventa: interpretarea lui "sa" exclusiv ca marker de subordonata, fara verificarea valorii verbale.
- Capcana frecventa: omisiunea subiectului subinteles in propozitii cu predicat la persoana I/II.
- Exercitiu recomandat: extrage dintr-un text de 15 randuri toate subordonatele si clasifica-le pe tipuri.
- Exercitiu recomandat: corecteaza 12 enunturi cu virgule gresite si explica regula in maximum o fraza.

**Exemple clare**
- Exemplu 1: "Stiu ca vei veni." -> "ca vei veni" este completiva directa (ce stiu?).
- Exemplu 2: "Cartea pe care ai recomandat-o este excelenta." -> "pe care ai recomandat-o" este atributiva (determina "cartea").
- Exemplu 3: "Daca inveti constant, progresezi." -> subordonata circumstantiala conditionala.
- Exemplu 4: "A muncit mult, deci a reusit." -> coordonare concluziva.
- Exemplu 5: "Elevul este pregatit." -> predicat nominal ("este pregatit").
- Exemplu 6: "Elevul redacteaza raspunsul corect." -> predicat verbal ("redacteaza").
- Exercitiu aplicat 1: explica de ce virgula este corecta sau incorecta in 8 enunturi cu subordonate diferite.
- Exercitiu aplicat 2: identifica in 5 minute toate predicatele dintr-un text de 12-15 randuri.
- Exemplu 7: "Desi era obosit, a continuat sa invete." -> subordonata concesiva + completiva directa.
- Exemplu 8: "A invatat atat de bine, incat a rezolvat rapid testul." -> subordonata consecutiva.
- Exercitiu aplicat 3: transforma 10 propozitii simple in fraze cu subordonate corecte si punctuatie justificata.

**Checklist rapid (pentru utilizator)**
- [ ] Delimitez corect propozitiile dintr-o fraza la prima lectura.
- [ ] Identific raportul principal (coordonare/subordonare) fara ezitare.
- [ ] Diferentiez atributiva de completiva directa in exemple noi.
- [ ] Pot justifica punctual fiecare virgula dintr-o fraza complexa.
- [ ] Rezolv corect teste de sintaxa in timp limitat.
