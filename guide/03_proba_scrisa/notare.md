# Notare si criterii de promovare

**Rezumat**
Nota la proba scrisa se calculeaza pe disciplina. Fiecare disciplina trebuie promovata cu minimum 5.

**Detalii**
- Punctajele se transforma in note prin formula din regulament.
- Nota finala la proba scrisa este media aritmetica a notelor pe discipline.
- Pentru promovare, fiecare disciplina trebuie sa aiba minimum nota 5.

**Surse**
- Conform regulamentului din `Regulament-admitere-licenta-2026-actualizare-18-01-2026.pdf`
