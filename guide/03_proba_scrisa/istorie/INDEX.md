# Istorie

**Rolul sectiunii**
Sectiunea de istorie acopera temele oficiale pentru proba scrisa, pe procese istorice, contexte si relatii cauzale.

**Capitole**
- Popoare si spatii istorice
- Oamenii, societatea si lumea ideilor
- Statul si politica
- Relatiile internationale

**Regula de utilizare**
- Incepe cu sinteza cronologica.
- Continua pe capitole tematice si apoi pe teste grila.

**Ultima actualizare**
- 2026-02-11
