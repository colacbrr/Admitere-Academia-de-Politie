# Istorie

**Rolul sectiunii**
Sectiunea de istorie construieste gandirea istorica necesara pentru examen: definitii precise, context cronologic, relatii cauza-efect, comparatie intre regimuri si interpretare critica a surselor.

**Capitole**
- Popoare si spatii istorice
- Oamenii, societatea si lumea ideilor
- Statul si politica
- Relatiile internationale
- Constitutiile Romaniei (1866-2003)
- Razboiul Rece - studii de caz

**Metoda de parcurgere**
- Incepi cu conceptele de baza si cronologia minima.
- Continui cu analize tematice pe capitole.
- La final, exersezi raspunsuri argumentate scurte.

**Ce urmarim la invatare**
- Claritate conceptuala.
- Corelare concept -> perioada -> exemplu.
- Argumentare istorica riguroasa.
- Retentie pe termen lung prin recapitulare activa.
- Folosirea corecta a conceptelor de istoriografie, periodizare si cauzalitate istorica.
- Diferentierea dintre fapt istoric, interpretare istorica si sursa istorica.

**Standard de calitate pe capitol**
- Capitolul este \"gata\" la prag minim 90%: concept definit clar, context temporal explicit, exemple istorice corecte, capcane frecvente acoperite, exercitii aplicate.
- Pentru performanta constanta, fiecare capitol trebuie revizitat prin repetitie activa la 24h, 7 zile si 30 zile.
