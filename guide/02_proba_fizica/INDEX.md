# Proba fizica

**Rolul modulului**
Acest modul explica etapa fizica a admiterii: cum este evaluata, ce reguli se aplica, cum arata traseul si cum te pregatesti eficient.

**Prioritate pentru utilizator**
- Intelegi ce inseamna etapa eliminatorie.
- Stii ce pregatire minima este necesara pentru promovare.

**Capitole**
- Descriere proba
- Reguli si criterii de promovare
- Traseu pe puncte
- Pregatire fizica

**Regula de utilizare**
- Incepe cu descrierea probei.
- Continua cu reguli/criterii si apoi traseul.
- Foloseste sectiunea de pregatire pentru planificare.
