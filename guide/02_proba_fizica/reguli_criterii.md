# Reguli si criterii de promovare

**Rezumat**
Proba este evaluata prin calificativ si nota, conform anexelor din regulament.

**Detalii**
- Nota se incadreaza intre 5,00 si 10,00.
- Criteriile si baremele sunt in anexele regulamentului.

**Surse**
- Conform regulamentului din `Regulament-admitere-licenta-2026-actualizare-18-01-2026.pdf`
