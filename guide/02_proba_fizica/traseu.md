# Traseu pe puncte

**Identificare**
- Modul: `02_proba_fizica`

- Ultima verificare surse: `2026-02-11`

**Ce trebuie sa invete candidatul**
- Structura traseului si ordinea obstacolelor.
- Reguli de executie pentru fiecare element.
- Tipurile de penalizari sau motive de nepromovare.

**Ce trebuie sa pregateasca candidatul**
- Antrenament orientat pe obstacolele traseului.
- Tehnica de executie pentru fiecare secventa.
- Verificare finala: rulare completa a traseului in ordinea oficiala.

**Rezumat**
Traseul este definit in anexele regulamentului, cu obstacole, ordine de executie si reguli de penalizare clare.

**Explicatie detaliata**
- Obstacolele si ordinea lor sunt stabilite in anexa de specialitate.
- Pentru fiecare element exista conditii de executie validate prin barem.
- Penalizarile si situatiile de eliminare se aplica strict conform regulamentului.

**Exemple clare**
- Exemplu 1: candidatul respecta ordinea obstacolelor si finalizeaza traseul in limitele admise.
- Exemplu 2: candidatul rateaza un element critic si primeste rezultat nepromovat conform regulilor.

- [ ] Cunosc ordinea oficiala a traseului.
- [ ] Cunosc regulile de executie pe obstacole.
- [ ] Cunosc penalizarile si situatiile de eliminare.
- [ ] Pot executa integral traseul in antrenament.
