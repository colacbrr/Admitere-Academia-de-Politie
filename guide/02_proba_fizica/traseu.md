# Traseu pe puncte

**Rezumat**
Traseul este descris detaliat in anexe. Ghidul va extrage fiecare obstacol si regula asociata.

**Detalii**
- Obstacolele si ordinea lor sunt stabilite in anexa de specialitate.
- Orice penalizare sau eliminare se aplica conform regulamentului.

**Surse**
- Conform regulamentului din `Regulament-admitere-licenta-2026-actualizare-18-01-2026.pdf`
