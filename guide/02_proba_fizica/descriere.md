# Descriere proba

**Rezumat**
Proba fizica este prima etapa a concursului si este eliminatorie.

**Detalii**
- Candidatii se prezinta cu act de identitate si echipament sportiv.
- Rezultatul este "promovat" sau "nepromovat".
- Nu se admit reexaminari sau contestatii la proba fizica.

**Surse**
- Conform regulamentului din `Regulament-admitere-licenta-2026-actualizare-18-01-2026.pdf`
