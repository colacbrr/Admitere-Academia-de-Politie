# Inscriere si dosar

**Rezumat**
Inscrierea se face prin unitatile de recrutare. Dosarul include documente standard de identitate, studii si avize.

**Detalii**
- Dosarele se constituie conform normelor MAI.
- Inscrierea este conditionata de validarea documentelor si a taxei sau scutirii.

**Surse**
- Conform regulamentului din `Regulament-admitere-licenta-2026-actualizare-18-01-2026.pdf`
