# Observatii si actualizari

**Identificare**
- Modul: `05_calendar`
- Capitol: `actualizari`
- Ultima actualizare: `2026-02-11`
- Ultima verificare surse: `2026-02-11`
- Autor revizie: `echipa proiect`

**Ce trebuie sa invete candidatul**
- Calendarul poate suferi modificari oficiale.
- Singura referinta valida este comunicarea oficiala a Academiei.
- Diferenta dintre versiunea de lucru interna si anuntul oficial public.

**Ce trebuie sa pregateasca candidatul**
- Verificare periodica a site-ului oficial de admitere.
- Actualizare imediata a planului personal la orice schimbare.
- Verificare finala: reconfirmare cu 24-48h inainte de fiecare termen critic.

**Rezumat**
Calendarul se poate actualiza anual sau punctual, iar ghidul trebuie sa afiseze explicit anul si data ultimei verificari.

**Explicatie detaliata**
- Se verifica periodic pagina oficiala de admitere pentru anunturi noi.
- Daca apar modificari, se actualizeaza doar sectiunile afectate, cu marcarea datei.
- Pentru orice neconcordanta, prevaleaza informatia publicata oficial.

**Exemple clare**
- Exemplu 1: se modifica data unei probe, iar candidatul isi ajusteaza imediat planul de pregatire.
- Exemplu 2: apare un anunt suplimentar privind inmatricularea; se actualizeaza capitolul cu data exacta.

**Checklist rapid (pentru utilizator)**
- [ ] Verific site-ul oficial saptamanal in perioada premergatoare.
- [ ] Verific zilnic in saptamana etapelor.
- [ ] Actualizez planul personal dupa fiecare comunicare oficiala.
- [ ] Verific data ultimei actualizari din ghid.
