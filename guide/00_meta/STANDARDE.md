# Standarde de continut

**Format**
- Titluri scurte si consistente.
- Liste fara sub-bullets.
- Link-uri scrise in format cod: `https://...`
 - Sunt permise tabele si figuri pentru claritate.

**Citari**
- Informatiile din regulament se mentioneaza ca "Conform regulamentului".
- Informatiile din exterior trebuie citate explicit cu link.

**Unitate de continut**
Fiecare capitol trebuie sa aiba:
- Rezumat scurt
- Detalii
- Surse

**Note**
- Daca o informatie este valida doar pentru o sesiune (ex: 2026), se specifica anul.
