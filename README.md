# Admitere-Academia-de-Politie

Platforma pentru pregatirea admiterii la Academia de Politie.

## Prioritati

- Prioritatea 1: continut de studiu (materii, subiecte, explicatii detaliate, exemple, surse citate).
- Prioritatea 2: informatii organizatorice despre inscriere (etape, acte, termene, conditii).
- Continutul se extinde continuu cu informatii noi relevante, doar din surse credibile verificabile.
